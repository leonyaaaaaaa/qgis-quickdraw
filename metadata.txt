changelog=2.1.0
    - Quick tag picker reworked: now steps through your groups one at a time
      in order (Tab skips a group, Enter confirms it and moves to the next);
      finishes automatically after the last group
    - Multiple tags from the same group can be combined in one field
      (e.g. Au, As, Ag together)
    - Per-group "Case sensitive" option for letter-keyed groups
2.0.0
    - Icons switched from compiled Qt resources to SVG files loaded directly from disk
    - Qt6/QGIS4 compatibility fixes (QDesktopWidget removal, dead PyQt4-era code removed)
    - New: choose Shapefile (on disk) instead of memory layer as the output format
    - New: quick tag picker - press Tab in the note field to pick tags from up to 10
      configurable groups (numeric 1-9,0 or custom letter keys); adds a Tags column
    - Compatibility range set to QGIS >=3.0.0, <=4.9.9
